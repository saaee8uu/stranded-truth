Geography is evidence

MAP ROOM

Maps will appear only when backed by geographic data. Every layer records year, unit, source and uncertainty.

LAYER 001

Birthplace and public systems

Planned layers: district health, child nutrition, maternal mortality, school access, internet access and mobility. No map is published until the underlying geographic file and source table are archived.

MAP RULE

Never confuse area with people

State and district averages describe distributions, not individuals. Choropleths can exaggerate large areas and hide population density; every published map must state that limitation.
